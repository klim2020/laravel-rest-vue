# Small Laravel+Vuejs Starter kit

Client - Server with API 

### Back-end url

```https://my-portfolio2022-klim.herokuapp.com/```

### Front-End url

```https://klim2020.github.io/```

### Run sail container
```sh sl up -d```

### Deploy Frontend

```cd resources/render/apivue/```
```sh sl run deploy```

### Deploy Backend

```heroku login```

```git push heroku master:main```
